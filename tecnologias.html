Tecnologias Web

Minhas principais ferramentas para desenvolvimento front-end

HTML5

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. Ele estrutura o conteúdo da página usando elementos como títulos, parágrafos, links e imagens.

CSS3

CSS (Cascading Style Sheets) é usado para estilizar os elementos HTML. Com ele, podemos controlar cores, tamanhos, layouts responsivos e animações visuais.

JavaScript

JavaScript é uma linguagem de programação que permite adicionar interatividade às páginas da web, como sliders, menus dinâmicos, validação de formulários e muito mais.
